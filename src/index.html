Annual Strategic Plan JSON Conversion Tool

Data Center Optimization Initiative (DCOI)

Introduction

In preparation for the forthcoming OMB DCOI Strategic Plan, the following conversion tool will support Agencies’ ability to prepare JSON file submissions. Strategic Plans then must be published in a machine-readable JSON format at ‘agency’.gov/digitalstrategy under a section entitled, “Data Center Optimization Initiative Strategic Plans.”

Thank you for taking the time to enter the information below.

Guidance

Please enter the relevant data into each box. Note that some boxes will only accept numerical data. In the event that the data does not match the parameters for a specific box, an error statement will appear below describing what sort of data is required to fix the error. After filling in the data, click the submit button at the bottom. The tool will produce the entered data in a JSON format in a box at the very end. Please copy this output into a text file and save it in a JSON format. For additional guidance, please visit the management.cio.gov/schema.

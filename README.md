# DCOI-json-editor

## Install local/dev

    git clone git@github.com:REI-Systems/OMB-FITARA-DCOI-json-editor.git
    cd OMB-FITARA-DCOI-json-editor
    npm i
    npm start

Open http://localhost:3000/ in your browser

### Push to GitHub Pages

    npm run publish-to-gh
